CSS stands for Cascading Style Sheets. It is a styling language used alongside html to make webpages look more beautiful and appealing.There are three types or methods of CSS namely In line CSS sytling, internal CSS styling and external CSS styling.
The In line CSS is used within the opening tag of an HTML Element. To style a HTML file using Internal CSS, a style tag has to be used within the head element just after the title tag.
The External CSS involves making use of an external CSS file to style the HTML page. The CSS file is linked to the html page using a link in the head section of the HTML page. Also CSS involves the use of 5 selectors namely: Universal, Element, ID, Class and Class Attribute selector.

In CSS styling , it is very important to know the difference between paddings, margins and borders.The CSS Box Model controls Paddings, Borders and Margins.
Paddings are used to generate or create a space between an element's content and its borders.
Margins on the other hand are the spaces outside an element's border. The Margin controls the space
outside an element while padding controls the space within an element.
A Border is used to create a visible line or frame around the element it is applied to and it is used to set a border on an element.
